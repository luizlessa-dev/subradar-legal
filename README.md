# SubRadar — Páginas Legais e de Suporte

Documentos legais, políticas e páginas de suporte do **SubRadar**, radar inteligente de assinaturas recorrentes.

## Sobre o SubRadar

O SubRadar ajuda usuários a rastrear e gerenciar todas as suas assinaturas recorrentes em um só lugar, evitando cobranças esquecidas e gastos desnecessários.

## Páginas

| Arquivo | Descrição |
|---|---|
| `index.html` | Índice de documentos legais |
| `legal.html` | Termos de uso e Política de privacidade |
| `cookies.html` | Política de cookies |
| `delete-account.html` | Solicitação de exclusão de conta (LGPD) |
| `support.html` | Central de suporte ao usuário |

## Por que um repo separado?

As páginas legais ficam em repositório público para garantir transparência com os usuários e facilitar referência direta via URL. O app principal do SubRadar é privado.

## Deploy

Site estático hospedado via Vercel. As páginas ficam acessíveis em URLs públicas referenciadas dentro do app.
